<h2><a href=https://www.codewars.com/kata/54147087d5c2ebe4f1000805/train/javascript target="_blank">The 'if' function</a></h2><h3>8 kyu</h3><p>Create a function that takes three arguments:</p><ol><li>a value to be evaluated for truthiness.</li><li>a function to execute if the first argument is truthy.</li><li>a function to execute if the first argument is falsy.</li></ol><p>If the first argument evaluates to <em>truthy</em>, call the second argument (a function).If it evaluates to <em>falsy</em>, call the third argument instead (also a function).</p><p>Note: The first argument can be any value, since most languages allow any value to be interpreted as truthy or falsy. In strongly typed languages, however, it will typically be a <code>boolean</code>.</p>
